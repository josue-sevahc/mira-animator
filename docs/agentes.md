# Agents

Every Mira agent is a Claude skill: invoke it with its `/name`, or just describe what you want and the agent triggers itself. For how they connect, see the [Agent pipeline](pipeline.md).

The agents are grouped into five families — each has its own page:

- **[Core agents](agentes/core.md)** — the heart of deck creation: requirements, references, animation, sizing, images and videos.
- **[Utility agents](agentes/uteis.md)** — the content chain: extract, plan, write copy and validate.
- **[Visuals & data](agentes/visuais.md)** — static visuals, charts, QR codes, true 3D and image-driven templates.
- **[Responsive agents](agentes/responsivos.md)** — square, vertical and rule-of-thirds versions of a deck.
- **[Transitions](agentes/transicoes.md)** — slide-to-slide transition effects.
